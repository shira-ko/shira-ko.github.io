introduction: この講義の目的

目的

  1. プログラミングの初学的講座
  2. 受講後に，同様の開発環境を再現できるようになる
  3. 自主学習を進められる自力をつける
  4. インターネットで有用な情報を得るための検索クエリの目星を立てられる
  5. プログラミングをやったことがない人に紹介できる

使用する言語とフレームワーク

  • p5.js (javaScript)
  • Arduino IDE (C)

講座を受ける心構え

  • 講座に能動的に"参加"しよう！
  • わからないことがあったら即質問しよう！
  • わかったところは積極的に周りにシェアしよう！
  • 顔をしかめてもわかるようになることはないので，わからないを楽しもう！
  • エラー（失敗）はもっとも良い学び！繰り返すことで理解していこう！
